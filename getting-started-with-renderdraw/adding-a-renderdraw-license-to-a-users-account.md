# Adding a RenderDraw License to a user's account

{% embed url="https://app.tango.us/app/workflow/Adding-a-RenderDraw-license-to-a-user-756c4219592d463c928bd8c81b8730fc" %}

## [Adding a RenderDraw license to a user](https://app.tango.us/app/workflow/756c4219-592d-463c-928b-d8c81b8730fc?utm_source=magicCopy\&utm_medium=magicCopy\&utm_campaign=workflow%20export%20links)

To use RenderDraw components, a user (either standard Salesforce or Experience cloud) must have a user license installed. Let's take a look at how to add this license to a user's account.



***

#### 1. Start in the Setup screen

#### 2. Type "installed" in the search window

![Type "installed" in the search window](https://images.tango.us/workflows/756c4219-592d-463c-928b-d8c81b8730fc/steps/99adfc4f-fdd3-4bde-b093-b5895bf3f2ae/6cea0152-643d-4d5d-a8a6-c1c74ceee5e2.png?fm=png\&crop=focalpoint\&fit=crop\&fp-x=0.2500\&fp-y=0.2498\&fp-z=2.0000\&w=1200\&ar=2624%3A1476)

#### 3. Click on Installed Packages

![Click on Installed Packages](https://images.tango.us/workflows/756c4219-592d-463c-928b-d8c81b8730fc/steps/f338e969-a108-4390-b3f4-f40f7242641d/8166410a-0920-4f63-b2d7-33d9a4ca2154.png?fm=png\&crop=focalpoint\&fit=crop\&fp-x=0.2499\&fp-y=0.2498\&fp-z=2.0000\&w=1200\&ar=2624%3A1476)

#### 4. Locate RenderDraw's package, and Click on Manage Licenses

![Locate RenderDraw's package, and Click on Manage Licenses](https://images.tango.us/workflows/756c4219-592d-463c-928b-d8c81b8730fc/steps/ea0c2d17-4faf-4773-9fa7-3bf01a40e470/1ec9404e-228a-400d-82a0-e097b8cfc877.png?fm=png\&crop=focalpoint\&fit=crop\&fp-x=0.5000\&fp-y=0.5000\&fp-z=1.0000\&w=1200\&ar=2624%3A1476)

#### 5. Click on add Users

![Click on add Users](https://images.tango.us/workflows/756c4219-592d-463c-928b-d8c81b8730fc/steps/cdbc57f8-a45b-4f04-94e4-2151862bdc02/93919a15-0e3b-4496-8848-2e1f53361667.png?fm=png\&crop=focalpoint\&fit=crop\&fp-x=0.3642\&fp-y=0.4090\&fp-z=2.0000\&w=1200\&ar=2624%3A1476)

#### 6. Check the box next to the Available user's record to add the license

![Check the box next to the Available user's record to add the license](https://images.tango.us/workflows/756c4219-592d-463c-928b-d8c81b8730fc/steps/65ebf745-4658-4219-a669-e95b983a4b89/2b4978b5-ab29-4f33-8880-d2bb8a18328f.png?fm=png\&crop=focalpoint\&fit=crop\&fp-x=0.5000\&fp-y=0.5000\&fp-z=1.0000\&w=1200\&ar=2624%3A1476)

#### 7. Submit Add Users…

![Submit Add Users…](https://images.tango.us/workflows/756c4219-592d-463c-928b-d8c81b8730fc/steps/4a2d39ea-1371-409c-9351-7f0d66b0c57d/097bbdaa-f9b4-4c93-bd9d-9fddb556deff.png?fm=png\&crop=focalpoint\&fit=crop\&fp-x=0.5000\&fp-y=0.5000\&fp-z=1.0000\&w=1200\&ar=2624%3A1476)

#### 8. Excellent! Ready to add permission sets and start using RenderDraw
