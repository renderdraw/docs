# Installation

### Install

1. To install the latest version, contact a RenderDraw representative [here](https://calendly.com/d/dqm-zjb-nrh/renderdraw-intro-call)
   1. Package install is standard from Salesforce AppExchange.&#x20;
      1. If an earlier version exists, upgrading is the same process
2. Install for **Admins only**, and wait for completion
   1. **By default, RenderDraw is enabled for 14 days as a trial**
      1. An extension request can be made below

### Apply Permission Sets

1. Once installed, ensure you add the proper permissions for your users
   1. The **RenderDraw User** permission set allows _view and interact-only_ access using the Simple Renderer Component
   2. The **RenderDraw Admin** **User** permission set allows for _the setup of relationships, AdvancedRenderer Component, and 3D Scene Setup_
   3. To learn more about permission sets from Salesforce, [click here](https://trailhead.salesforce.com/content/learn/modules/data_security/data_security_objects)
   4. To view all RenderDraw related permission sets, [visit this page](renderdraw-permission-sets/)
