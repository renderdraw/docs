# Utilizing Interactions within Lightning Flow

Lightning flows enable RenderDraw components to be used in a broad array of use cases within the Salesforce platform. Essentially Screen flows can be used anywhere, so this is how we often integrate with different aspects of the platform.

The majority of our components are enabled for Flow interaction and often require the same parameters for providing visual interaction as simply adding the appropriate component within your Screen Flow. Let's take a step-by-step look here.
