# Salesforce CPQ

The integration between RenderDraw and Salesforce CPQ is delivered via managed package and installed and used through the product configuration flow.

**Prerequisites:**

* RenderDraw package is [installed](../../../../../../../getting-started-with-renderdraw/installation.md) and ready to use within product records
* Salesforce CPQ is installed, and configured for use (catalog, pricing, etc.)
* External Configuration is enabled for products being visualize via RenderDraw
