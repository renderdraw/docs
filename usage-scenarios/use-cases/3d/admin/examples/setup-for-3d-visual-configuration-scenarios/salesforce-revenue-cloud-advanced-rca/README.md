# Salesforce Revenue Cloud Advanced (RCA)

The integration between RenderDraw and Salesforce RCA is delivered via managed package and installed and used through the product configuration flow.

**Prerequisites:**

* RenderDraw main package **and** RCA add-on package are [installed](../../../../../../../getting-started-with-renderdraw/installation.md) and ready to use within product records
* Salesforce RCA is installed, and configured for use (catalog, pricing, etc.)

**Tasks:**

* Add 2D/3D Visual Scene Setup component to Product layout
* Extend the Product Configuration Flow,&#x20;
* Add RenderDraw for Revenue Cloud component to flow&#x20;
* Connect properties from data manager
* Assign newly created Product configuration flow to products with visuals
* Use the Visual Logic Generator button to generate the visual logic based on the Revenue Cloud data model
