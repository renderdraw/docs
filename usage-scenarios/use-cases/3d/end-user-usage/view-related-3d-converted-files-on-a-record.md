### # In your Salesforce org, add the 3D File Viewer component to your Lightning page

#### 4. To add more native file types to the record for viewing, Click on Upload Files…

This works for GLB, GLTF, STL, and OBJ files by default. Convert your CAD files if needed before doing this using the renderdraw conversion API
