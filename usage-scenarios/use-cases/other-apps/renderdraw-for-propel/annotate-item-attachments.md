# Annotate Item Attachments

Additionally, if updated with text, the text will get posted to the related record's Chatter feed with the configured settings

## [Annotate Item Attachments - RenderDraw for PropelPLM](https://app.tango.us/app/workflow/7fc038e9-79bd-4d16-8451-6fe4932b75cb?utm_source=magicCopy\&utm_medium=magicCopy\&utm_campaign=workflow%20export%20links)

#### 1. Go to Item details

#### 2. Click on Show more actions

![Click on Show more actions](https://images.tango.us/workflows/7fc038e9-79bd-4d16-8451-6fe4932b75cb/steps/549dbf9a-3e17-4422-83c7-292c57c0d706/ee96e0e1-e0d4-4fd3-86da-9ecb6579d0e0.png?fm=png\&crop=focalpoint\&fit=crop\&fp-x=0.9680\&fp-y=0.1728\&fp-z=3.0196\&w=1200)

#### 3. Click on Annotate Attachments

![Click on Annotate Attachments](https://images.tango.us/workflows/7fc038e9-79bd-4d16-8451-6fe4932b75cb/steps/1619c952-bc07-4ef0-aaa9-0ce71e7b6808/978f49e7-0321-4525-9396-d9c91353da6c.png?fm=png\&crop=focalpoint\&fit=crop\&fp-x=0.8958\&fp-y=0.4580\&fp-z=2.9830\&w=1200)

#### 4. Select Item Revision

![Select Item Revision](https://images.tango.us/workflows/7fc038e9-79bd-4d16-8451-6fe4932b75cb/steps/bf0ab014-70c2-457d-9497-f319820c2768/c6235b9f-df88-4770-9baf-2367213d8a54.png?fm=png\&crop=focalpoint\&fit=crop\&fp-x=0.4998\&fp-y=0.4817\&fp-z=1.3943\&w=1200)

#### 5. Click on Next

![Click on Next](https://images.tango.us/workflows/7fc038e9-79bd-4d16-8451-6fe4932b75cb/steps/0b8b675f-5fed-48e6-9614-4c3e95914ba3/31d2f96a-55d7-4787-81f9-efbe03d5da05.png?fm=png\&crop=focalpoint\&fit=crop\&fp-x=0.8133\&fp-y=0.6050\&fp-z=2.8233\&w=1200)

#### 6. Select the Document Attachment you want to edit

This search is limited to PDF, PNG, or JPG files.&#x20;

![Select the Document Attachment you want to edit](https://images.tango.us/workflows/7fc038e9-79bd-4d16-8451-6fe4932b75cb/steps/b98b6e56-b4ec-4eae-b31a-00d255e8810d/c53f0eab-5cfa-446b-aaf7-558afc4a133c.png?fm=png\&crop=focalpoint\&fit=crop\&fp-x=0.4998\&fp-y=0.4817\&fp-z=1.3943\&w=1200)

#### 7. Click on Annotate Document

![Click on Annotate Document](https://images.tango.us/workflows/7fc038e9-79bd-4d16-8451-6fe4932b75cb/steps/e3ae15ed-36c1-4c5a-9b98-c6b6a6d7595f/e020dfa9-96c1-45ee-a46d-508f1477a8a3.png?fm=png\&crop=focalpoint\&fit=crop\&fp-x=0.7778\&fp-y=0.6050\&fp-z=2.8233\&w=1200)

#### 8. Now that the editor is open, Annotate the image by cropping, annotating, redacting, or resizing. Click on Annotate

![Now that the editor is open, Annotate the image by cropping, annotating, redacting, or resizing. Click on Annotate](https://images.tango.us/workflows/7fc038e9-79bd-4d16-8451-6fe4932b75cb/steps/428cbfeb-5240-4c54-9a3d-139caf39ccad/4b6ba485-7b4d-4390-9aa7-a8aad865f51a.png?fm=png\&crop=focalpoint\&fit=crop\&fp-x=0.2233\&fp-y=0.5650\&fp-z=2.6367\&w=1200)

#### 9. To add text, Click on Text. This will update the chatter feed text for the parent Item

![To add text, Click on Text. This will update the chatter feed text for the parent Item](https://images.tango.us/workflows/7fc038e9-79bd-4d16-8451-6fe4932b75cb/steps/b50bb8de-a998-4bd4-a5a5-6a48fc9e9156/c3e74fcc-ab5b-475b-a0d3-d79c2c94dfdd.png?fm=png\&crop=focalpoint\&fit=crop\&fp-x=0.7654\&fp-y=0.7019\&fp-z=3.2087\&w=1200)

#### 10. Add any additional annotations desired

![Add any additional annotations desired](https://images.tango.us/workflows/7fc038e9-79bd-4d16-8451-6fe4932b75cb/steps/1bb19275-8db6-4aef-85c9-1b0b48f43cde/1b427381-0816-4c95-8510-ccf462fdc98a.png?fm=png\&crop=focalpoint\&fit=crop\&fp-x=0.5000\&fp-y=0.5000\&w=1200)

#### 11. Click on Done to save the image

![Click on Done to save the image](https://images.tango.us/workflows/7fc038e9-79bd-4d16-8451-6fe4932b75cb/steps/c5269e8f-912a-4d78-bf8a-21ca5a2cf814/5cda83dd-69ee-424e-83d3-551a8707d464.png?fm=png\&crop=focalpoint\&fit=crop\&fp-x=0.7929\&fp-y=0.2703\&fp-z=2.9124\&w=1200)

#### 12. Close the Modal

![Close the Modal](https://images.tango.us/workflows/7fc038e9-79bd-4d16-8451-6fe4932b75cb/steps/b7a2cb94-3e3a-408d-b8cb-a234585506a3/7050fd88-5cce-44ec-a5cb-b3cf4bd83d88.png?fm=png\&crop=focalpoint\&fit=crop\&fp-x=0.8438\&fp-y=0.0888\&fp-z=2.9124\&w=1200)

#### 13. Click on Refresh this feed to see the update

![Click on Refresh this feed to see the update](https://images.tango.us/workflows/7fc038e9-79bd-4d16-8451-6fe4932b75cb/steps/537d2183-ecae-44d0-bfe7-cff8ec458fb1/eb651180-a96d-47ef-b48d-1de3e936d7a5.png?fm=png\&crop=focalpoint\&fit=crop\&fp-x=0.9672\&fp-y=0.5400\&fp-z=2.8233\&w=1200)

#### 14. The annotation is now added to the Chatter feed

![The annotation is now added to the Chatter feed](https://images.tango.us/workflows/7fc038e9-79bd-4d16-8451-6fe4932b75cb/steps/a0c64bbb-7024-4fe0-85bd-5ea3b96e2aed/a876624b-1098-49b7-aeb0-3e4be1a95297.png?fm=png\&crop=focalpoint\&fit=crop\&fp-x=0.8352\&fp-y=0.9512\&fp-z=2.4000\&w=1200)
