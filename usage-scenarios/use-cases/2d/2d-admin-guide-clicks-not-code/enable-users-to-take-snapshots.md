## [2D Admin - Enable users to take a snapshot](https://app.tango.us/app/workflow/019bbdb5-efdf-4fec-9143-3ee243f1da89?utm_source=magicCopy\&utm_medium=magicCopy\&utm_campaign=workflow%20export%20links)

#### 1. Go to the page where the Admin 2D component has been added

#### 2. Go to the environment settings of the scene

![Go to the environment settings of the scene](https://images.tango.us/workflows/019bbdb5-efdf-4fec-9143-3ee243f1da89/steps/3aac57f7-18f1-43a6-99d8-2a22dfa33c67/44bf79be-8ba7-4788-a364-f48b609a2383.png?fm=png\&crop=focalpoint\&fit=crop\&fp-x=0.1784\&fp-y=0.5044\&fp-z=1.6743\&w=1200\&ar=2624%3A1476)

#### 3. Click on "Show all Environment Options" button to see all options

![Click on "Show all Environment Options" button to see all options](https://images.tango.us/workflows/019bbdb5-efdf-4fec-9143-3ee243f1da89/steps/c25da822-6eed-42d3-bc30-a309444c422d/f859bdb6-2372-4c0d-8746-efb7bd3f696f.png?fm=png\&crop=focalpoint\&fit=crop\&fp-x=0.1785\&fp-y=0.5278\&fp-z=1.6605\&w=1200\&ar=2624%3A1476)

#### 4. Go to the Menu Options; if it's not expanded, expand it to see all options

![Go to the Menu Options; if it's not expanded, expand it to see all options](https://images.tango.us/workflows/019bbdb5-efdf-4fec-9143-3ee243f1da89/steps/0a748c5e-7deb-4211-addb-4c349b43b283/9a79a24c-ba94-446e-8543-e66479b960ef.png?fm=png\&crop=focalpoint\&fit=crop\&fp-x=0.2342\&fp-y=0.4456\&fp-z=2.1102\&w=1200\&ar=2624%3A1476)

#### 5. Ensure the "Show Take Snapshot Button" is checked

![Ensure the "Show Take Snapshot Button" is checked](https://images.tango.us/workflows/019bbdb5-efdf-4fec-9143-3ee243f1da89/steps/e6c6cc2a-c9b5-446a-ab48-7a0f24ce26e0/c1749cde-0847-4e44-9589-ebb02502297d.png?fm=png\&crop=focalpoint\&fit=crop\&fp-x=0.1692\&fp-y=0.5075\&fp-z=3.1620\&w=1200\&ar=2624%3A1476)

#### 6. Go back to the main options context menu.

![Go back to the main options context menu.](https://images.tango.us/workflows/019bbdb5-efdf-4fec-9143-3ee243f1da89/steps/87475afb-5f84-4592-9b9f-5a4c9995f705/3d742b96-870b-44ab-befe-3edd90bb424b.png?fm=png\&crop=focalpoint\&fit=crop\&fp-x=0.0252\&fp-y=0.3662\&fp-z=3.1418\&w=1200\&ar=2624%3A1476)

#### 7. Click on Save

![Click on Save](https://images.tango.us/workflows/019bbdb5-efdf-4fec-9143-3ee243f1da89/steps/b787f26b-fc7d-4aff-9c4c-685cb87064cf/421f9252-e984-4854-a2e4-06fc61c5dbd7.png?fm=png\&crop=focalpoint\&fit=crop\&fp-x=0.3077\&fp-y=0.6799\&fp-z=2.9543\&w=1200\&ar=2624%3A1476)
