## [Create Droppable Area and Drop Zone](https://app.tango.us/app/workflow/7373d6cd-b7a4-4899-ad4e-c2f3a343f5b3?utm_source=magicCopy\&utm_medium=magicCopy\&utm_campaign=workflow%20export%20links)

For advanced 2D usage, such as configuration or creating a planogram, we need to have an element that can handle adding sub-objects or components into it. Enter the Droppable Area. The Droppable Area typically represents an assembly that would have suboptions that need to be selected, but it can also represent a shelf for retail, or warehouse storage racks for inventory management.&#x20;
