# Choose where you want to display your Product Catalogs

There are typically two different data models for attaching spare parts catalogs within a B2B

### Product Category Based Model

Setting up Product Catalogs based on the product category is a great way to utilize built in objects when starting with B2B LWR Digital Experiences.

**Pros**

* No need for Custom object creation

**Considerations**

* Catalogs must be visible as read only (at least) to your users to access custom fields on the object

There are times when it makes sense, especially if you already have custom objects or relationships to setup product catalog pages on Product detail pages or custom objects that represent Spare Parts hierarchies altogether.
