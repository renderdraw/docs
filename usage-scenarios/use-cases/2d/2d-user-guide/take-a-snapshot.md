# Take a snapshot

{% embed url="https://app.tango.us/app/embed/bbc1c6b4-bde7-461d-9da9-f56d157afce7?iframe=" %}

## [2D - Taking a snapshot](https://app.tango.us/app/workflow/bbc1c6b4-bde7-461d-9da9-f56d157afce7?utm_source=magicCopy\&utm_medium=magicCopy\&utm_campaign=workflow%20export%20links)

#### 1. Go to Your record on Salesforce page where the 2D component has been added.

Ensure the user has the RenderDraw User permission set

#### 2. Add your items to the screen either by data with the passed in Canvas, or by dragging and dropping.

![Add your items to the screen either by data with the passed in Canvas, or by dragging and dropping.](https://images.tango.us/workflows/bbc1c6b4-bde7-461d-9da9-f56d157afce7/steps/341d83cc-1d3c-4bd6-a659-1039bf7f4f42/8a7c9220-24ef-4860-813a-7fbaf89923fc.png?fm=png\&crop=focalpoint\&fit=crop\&fp-x=0.4998\&fp-y=0.3137\&fp-z=1.0178\&w=1200\&ar=2624%3A1476)

#### 3. Select "on"

![Select "on"](https://images.tango.us/workflows/bbc1c6b4-bde7-461d-9da9-f56d157afce7/steps/2541c707-8ca1-4850-a0ea-223d2df56949/45156dd2-13d7-4464-a075-70ad749407be.png?fm=png\&crop=focalpoint\&fit=crop\&fp-x=0.0175\&fp-y=0.4031\&fp-z=3.3333\&w=1200\&ar=2624%3A1476)

#### 4. Once ready to take a snapshot, view the toolbar on the 2D component

![Once ready to take a snapshot, view the toolbar on the 2D component](https://images.tango.us/workflows/bbc1c6b4-bde7-461d-9da9-f56d157afce7/steps/fbaa4adf-454a-4431-8ab7-0e29b738776a/73a843bc-6399-4155-ac7b-953f3261a47c.png?fm=png\&crop=focalpoint\&fit=crop\&fp-x=0.4998\&fp-y=0.3333\&fp-z=1.0178\&w=1200\&ar=2624%3A1476)

#### 5. Select the snapshot button

if this button isn't here, ensure your 2D scene settings were properly setup with the "show take snapshot button" selected in the environment options

![Select the snapshot button](https://images.tango.us/workflows/bbc1c6b4-bde7-461d-9da9-f56d157afce7/steps/6d14edf7-5065-4e03-a86e-f8ed32ec3ab0/9f2ab2a3-3dcf-4363-ab52-960cc68a24ce.png?fm=png\&crop=focalpoint\&fit=crop\&fp-x=0.1079\&fp-y=0.3327\&fp-z=3.1620\&w=1200\&ar=2624%3A1476)

#### 6. Once the preview dialog opens, either download or attach the image to a record

![Once the preview dialog opens, either download or attach the image to a record](https://images.tango.us/workflows/bbc1c6b4-bde7-461d-9da9-f56d157afce7/steps/d2bec06b-3ec2-4cb1-965b-c14b01b5e144/e409d7aa-2628-42fa-8710-db23fd51eac1.png?fm=png\&crop=focalpoint\&fit=crop\&fp-x=0.5000\&fp-y=0.5522\&fp-z=1.1328\&w=1200\&ar=2624%3A1476)

#### 7. Click on Attach the image to a record

![Click on Attach the image to a record](https://images.tango.us/workflows/bbc1c6b4-bde7-461d-9da9-f56d157afce7/steps/9024ddd0-52dc-438d-a2d9-fa1627b36e28/b8c5aea0-d913-4bd5-9ab9-41a32f7c61dd.png?fm=png\&crop=focalpoint\&fit=crop\&fp-x=0.5783\&fp-y=0.8757\&fp-z=2.2774\&w=1200\&ar=2624%3A1476)

#### 8. The updated dialog opens, this is where you can rename the file and select the file type

![The updated dialog opens, this is where you can rename the file and select the file type](https://images.tango.us/workflows/bbc1c6b4-bde7-461d-9da9-f56d157afce7/steps/4c106d54-1eaa-436c-8772-c71acecac8e1/1d71df2c-b4c1-49d0-b5ba-4102f9565925.png?fm=png\&crop=focalpoint\&fit=crop\&fp-x=0.5000\&fp-y=0.5268\&fp-z=1.5245\&w=1200\&ar=2624%3A1476)

#### 9. Find your record by searching for the object and name of the record to associate

![Find your record by searching for the object and name of the record to associate](https://images.tango.us/workflows/bbc1c6b4-bde7-461d-9da9-f56d157afce7/steps/ab59ca3d-9ff6-4c89-a646-ee82e92cf8c9/7bb165b7-4755-4bca-aa38-aa36141ea5f4.png?fm=png\&crop=focalpoint\&fit=crop\&fp-x=0.4950\&fp-y=0.4604\&fp-z=1.6916\&w=1200\&ar=2624%3A1476)

#### 10. Click on Add File to Record

![Click on Add File to Record](https://images.tango.us/workflows/bbc1c6b4-bde7-461d-9da9-f56d157afce7/steps/0f1e7bd4-cd53-4696-88e5-85a7cb3a37d9/ae3e20ac-400c-4b87-9a72-bfd6d07122ac.png?fm=png\&crop=focalpoint\&fit=crop\&fp-x=0.6330\&fp-y=0.6494\&fp-z=2.5746\&w=1200\&ar=2624%3A1476)

#### 11. A confirmation message will confirm the snapshot was saved

![A confirmation message will confirm the snapshot was saved](https://images.tango.us/workflows/bbc1c6b4-bde7-461d-9da9-f56d157afce7/steps/46ebc1b0-361e-4658-86fe-def74ccfd2ad/261ce779-e95f-4706-a11a-2087647c645a.png?fm=png\&crop=focalpoint\&fit=crop\&fp-x=0.4788\&fp-y=0.1301\&fp-z=2.0955\&w=1200\&ar=2624%3A1476)
